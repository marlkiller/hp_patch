## 🚀 启动方式

1. 正常打开 Hopper Disassembler 应用。
2. 如果遇到**Apple无法验证“Hopper Disassembler.app”是否包含可能危害Mac安全或泄漏隐私的恶意软件提示**,请运行以下命令解除应用的隔离标记：

```bash
/usr/bin/xattr -r -d com.apple.quarantine "/Applications/Hopper Disassembler v4.app"
```
